# DropFish v14.18 — Торговое судно

Новая механика:
- После последнего заброса в любую погоду, кроме Шторма, торговое судно прибывает с вероятностью 50%.
- При прямой поимке рыбы удочкой с вероятностью 40% дополнительно находится специальный предмет.
- Судно принимает 3 случайных вида предметов из 6.
- Каждый предмет обменивается на тяжеловеса (90%) или рыбу-гиганта (10%).
- Можно обменивать предметы по одному или все подходящие сразу.
- Итоги сессии, достижения и дата завершения появляются только после завершения торговли.
- Рыба от судна входит в итоговый вес и получает финальные усиления, но не считается прямым уловом и не атакуется Касаткой.

Основные параметры находятся в `BALANCE.tradeShip` в `script.js`.
