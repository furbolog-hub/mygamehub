ЗАГРУЗКА:
Загрузите всё содержимое этой папки в корень репозитория GitHub Pages.

ПУБЛИЧНЫЙ ЛИМИТ:
- Одна завершённая игровая сессия в сутки на данном браузере/устройстве.
- Сброс происходит в 00:00 по локальному времени устройства.
- При повторном входе показывается оставшееся время до сброса.

ЗАЩИТА КОДА:
- dropfish/script.js упакован и обфусцирован, читаемые значения веса не видны обычным поиском в DevTools.
- Исходные карты (source maps) не публикуются.
- Это усложняет бытовое редактирование, но клиентский JavaScript невозможно сделать полностью защищённым.
- Для честного рейтинга/призов генерацию и проверку результата необходимо переносить на сервер.
